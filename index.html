Stock Prediction

Project

The application made using the services of AWS predicts the opening and closing prices of stocks of various companies.

The need to be able to correctly predict prices of stocks of various companies is always a challenge for day and swift traders who totally depend on these price fluctuations in order to earn their basic living.
Therefore, an accurate model is required to predict their opening and closing prices.
The LSTM (Long Short-Term Memory) Networks in Machine Learning are an amazing way of predicting these prices. Primarily, they work due to two main reasons, first, they can store past information and secondly, they forget useless information. A combination of these two things results in a model which has proven to be very effective in sequence prediction problems.
LSTM has three main gates:

  • The Input gate: It adds information to the cell state.
  • The Forget gate: It removes the information that is no longer required by the model
  • The Output gate: Output Gate at LSTM selects the information to be shown as output

Services Used

The following services played a crucial role in making this project a success:

EC2

Amazon EC2 is used to generate a key pair which is further used to deploy our website using CodeStar

Sagemaker

The notebook instance of Sagemaker houses the python code for LSTM model which is used to predict the prices.

CodeStar

CodeStar is used to quickly build and deploy applications is used to deploy this webpage.

S3 Bucket

The data store of AWS which stores the database and files through which our model is trained and tested.

CodeCommit

The GitHub of AWS which is used to build the application (this webpage) and commit these changes.

AWS Amplify

AWS Amplify is used to connect to GitHub repository and then deploy the website.
